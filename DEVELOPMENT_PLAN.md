- 성경 CSV/JSON 데이터 파일

- Flutter HTTP 패키지 또는 gomobile 연동 예시

## ⚠️ 리스크 및 해결 전략

- Android에서 로컬 Go API 실행 문제 발생 가능 → FFI 방식 또는 백그라운드 서버 실행 구조 검토
- 벡터 인덱스 로딩 시 메모리 사용량 증가 → HNSW 기반 압축 및 부분 로딩 전략 적용
- 데이터 파일 크기 증가에 따른 배포 부담 → 초기 패키지 최소화 및 선택적 다운로드 제공
